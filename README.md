
# SleepGuard — ICT304 AI System Project

## Project Description
SleepGuard is a real-time AI-powered driver drowsiness 
detection system that uses a standard webcam to monitor 
driver eye state and trigger alerts when fatigue is detected.

## Team
- Person 1 — Webcam & Face Detection (camera.py)
- Person 2 — EAR Detection Module (ear.py)
- Person 3 — Alarm & Integration (alarm.py, main.py)

## How to Run
1. Install dependencies:
   pip install opencv-python mediapipe numpy
2. Run the prototype:
   python main.py
